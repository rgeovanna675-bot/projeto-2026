# Projeto Stelas-2026 

## Projeto
     O que queremos criar?

* Queremos criar um jogo em Python inspirado na Adedonha (Stop) e nele o jogador vai escolher um número de 1 a 26 e o programa vai transformar esse número em uma letra do alfabeto. Depois disso, o jogador terá que pensar em palavras que comecem com a letra sorteada para completar as categorias do jogo.

     Qual é o principal objetivo do nosso projeto?
  
* O principal objetivo do nosso projeto é criar um jogo divertido e fácil de usar que ajude a desenvolver o raciocínio rápido e o conhecimento de palavras. Além disso, queremos colocar em prática o que aprendemos sobre programação em Python durante o curso.

      Quais ferramentas vamos usar na nossa criação?
  
*Vamos usar a linguagem de programação Python para desenvolver o jogo e a plataforma Replit para escrever, testar e executar o código. Também vamos utilizar os recursos do próprio Python para criar a lógica do jogo e a interação com o jogador.
