# Projeto Stelas-2026  

## Projeto-https://codepen.io/Mariana-De-Faria/pen/VYPmGrP

   * O que queremos criar?
     
Queremos criar um jogo inspirado na Adedonha (Stop), desenvolvido em HTML, CSS e JavaScript e nele o jogador vai escolher um número de 1 a 26 e o programa vai transformar esse número em uma letra do alfabeto. Depois disso, o jogador terá que pensar em palavras que comecem com a letra sorteada para completar as categorias do jogo.

     * Qual é o principal objetivo do nosso projeto?
    
O principal objetivo do nosso projeto é criar um jogo divertido e fácil de usar que ajude a desenvolver o raciocínio rápido e o conhecimento de palavras. Além disso, queremos colocar em prática o que aprendemos sobre desenvolvimento web utilizando HTML, CSS e JavaScript durante o curso.

     * Quais ferramentas vamos usar na nossa criação?
      
Vamos usar as linguagens HTML, CSS e JavaScript para desenvolver o jogo e a plataforma CodePen para escrever, testar e executar o código diretamente no navegador. Também vamos utilizar os recursos dessas tecnologias para criar a lógica do jogo, a interface e a interação com o jogador.
